Width

El ancho de un elemento HTML, que se puede especificar en píxeles, porcentaje u otras unidades.

height

altura de un elemento HTML, que se puede especificar de manera similar al ancho.

background-color

El color de fondo de un elemento HTML, que se puede especificar utilizando un nombre de color, un código hexadecimal o un valor RGB.

margin

El margen exterior de un elemento HTML, que se puede especificar en píxeles, porcentaje u otras unidades.

Padding

El relleno interior de un elemento HTML, que se puede especificar de manera similar al margen.

color

El color del texto de un elemento HTML, que se puede especificar utilizando un nombre de color, un código hexadecimal o un valor RGB.

font-family

La fuente utilizada para el texto de un elemento HTML, que se puede especificar utilizando una familia de fuentes predeterminada o una fuente personalizada.

font-size

El tamaño de la fuente utilizada para el texto de un elemento HTML, que se puede especificar en píxeles, puntos u otras unidades.

text-align

La alineación del texto dentro de un elemento HTML, que se puede especificar como izquierda, derecha, centrado o justificado.
