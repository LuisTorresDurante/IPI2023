Reglas de cascada, Herencia y Especificidad

Reglas de cascada

Básicamente, cuando se aplican múltiples reglas CSS a un elemento, las reglas se clasifican según su importancia y se aplican en un orden determinado. Las reglas con mayor importancia se aplican primero, y luego se van aplicando las reglas con menor importancia en caso de que no se hayan especificado estilos para el elemento en cuestión.

Herencia

La herencia en CSS es el proceso por el cual ciertas propiedades de estilo se aplican a un elemento hijo a partir de su elemento padre. En otras palabras, si un elemento padre tiene un estilo aplicado a él, ese estilo se aplicará a sus elementos hijos, a menos que se especifique lo contrario.

Especificidad

La especificidad en CSS se refiere a la manera en que se determina qué regla de estilo se aplicará a un elemento HTML cuando varias reglas compiten por el mismo estilo. Cada regla de estilo tiene un nivel de especificidad que se basa en la combinación de selectores utilizados en ella. La especificidad se calcula sumando los valores asignados a los selectores en cada regla, y la regla con el valor más alto tendrá prioridad y se aplicará al elemento HTML correspondiente.
