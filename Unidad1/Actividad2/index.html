Servidores Web y de Aplicación

¿Qué es un servidor?

Para un servidor tenemos 2 definiciones, una de software y una de hardware.

  • Servidor (Hardware):
    Es una computadora que almacena el software de servidor web, y los archivos que componen un sitio web (por ejemplo, documentos HTML, imágenes, hojas de estilos CSS y archivos JavaScript). Un servidor web -hardware- se conecta a internet y mantiene el intercambio de datos con otros dispositivos conectados a la web.

  • Servidor (Software):
    Un servidor web tiene muchas partes que controlan cómo los usuarios de la web obtienen acceso a los archivos alojados en el servidor; es decir, mínimamente, un servidor HTTP. Un servidor HTTP es una pieza de software capaz de comprender URLs (direcciones web) y HTTP (el protocolo que tu navegador usa para obtener las páginas web). Un servidor HTTP puede ser accedido a través de los nombres de dominio de los sitios web que aloja, y entrega el contenido de esos sitios web alojados al dispositivo del usuario final.

Servidores web:

Los servidores web son software y hardware que utilizan los protocolos (como HTTP o SMTP) para responder a las peticiones hechas en la web.
Su función es mostrar, almacenar y procesar el contenido de una página web a los usuarios.

Estos servidores funcionan mediante la obtencion de un url por parte de un usuario a travez de un navegador, que luego es traducido para obtener la direccion IP del servidor web en el cual esta localizada la pagina web. El navegador solicita un archivo al servidor y este envia la pagina solicitada por el navegador mediante HTTP.

Servidores web populares:

  • Apache HTTP
  • Nginx
  • Microsoft Internet Information Services (IIS)

Servidores de aplicación:

Los servidores de aplicaciones son programas de servidor en una red distribuida que proporciona el entorno de ejecución para un programa de aplicación. Más específicamente, el servidor de aplicaciones es el componente de tiempo de ejecución principal en todas las configuraciones y donde una aplicación se ejecuta realmente.

  • WebSphere Application Server.
  • Domino.
  • Integrated Web Application Server
